# ⚙️ Dota Tools

A lightweight desktop utility for **Dota 2** players who share a PC across multiple Steam accounts. It lets you copy your Dota 2 config from one account to another in a single click, and shows each account's conduct stats right in the UI.

![Rust](https://img.shields.io/badge/built%20with-Rust-orange?logo=rust)
![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20Linux%20%7C%20macOS-blue)

---

## Features

- **Config copy** — copies all files from `570/remote/cfg` of the source account into the target account's cfg folder, creating it if it doesn't exist.
- **Conduct stats** — reads `latest_conduct_1.<id>.txt` and displays behavior score, rating badge (Good / Neutral / Bad / Very Bad), commend count, report count, abandoned matches, and comms reports.
- **Auto Steam detection** — finds the Steam `userdata` directory automatically:
  - Windows: reads the install path from the registry, then falls back to common locations; also scans every drive letter (`A:\` – `Z:\`) for Steam/SteamLibrary installations.
  - Linux/macOS: checks `~/.steam/steam`, `~/.local/share/Steam`, and common mount points.
  - Parses `libraryfolders.vdf` to cover games installed on secondary drives.
- **Remote profile enrichment** — fetches persona names and avatars from the Steam Community XML API for accounts that are missing local data. Downloads are cached locally so the network is only hit once per account.
- **Search/filter** — type-ahead search in both the Source and Target account lists.
- **Dark-themed GUI** — built with [egui](https://github.com/emilk/egui) / [eframe](https://github.com/emilk/egui/tree/master/crates/eframe).

## Requirements

- [Rust toolchain](https://rustup.rs/) (stable, 1.75+)
- Dota 2 installed via Steam
- Internet connection (optional — only needed to fetch profile names/avatars for accounts not yet cached)

> **Conduct stats note:** the conduct file is generated by Dota 2 on game launch. If it shows *"Conduct file not found"*, start Dota 2 once on that account and it will appear.

## Installation

```bash
git clone https://github.com/nukeddd/dota_tools
cd dota_tools
cargo build --release
```

The compiled binary will be at `target/release/dota_tools` (or `dota_tools.exe` on Windows).

---

## Usage

1. Run the application.
2. The app auto-detects your Steam installation and lists all accounts found in `userdata/`.
3. Select a **Source** account (the one whose config you want to copy *from*).
4. Select a **Target** account (the one you want to copy the config *to*).
5. Click **Copy Config →**.
6. The status bar confirms success or shows an error message.

Conduct stats for each selected account are displayed beneath the account picker so you can verify you have the right accounts before copying.

## How It Works

| Step | Detail |
|------|--------|
| Steam discovery | Registry lookup → common paths → full drive scan + `libraryfolders.vdf` parsing |
| Account listing | Enumerates numeric subdirectories inside `userdata/`, reads `localconfig.vdf` for persona names |
| Conduct stats | Parses `<dota_cfg>/latest_conduct_1.<id>.txt` with a key-value regex |
| Remote data | Async `reqwest` calls to `steamcommunity.com/profiles/<id64>?xml=1`; avatars cached to the OS cache dir |
| Config copy | `fs::copy` of every file under `userdata/<id>/570/remote/cfg/` |
